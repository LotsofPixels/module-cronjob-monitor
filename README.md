Cronjob Monitor
The monitor sends a heartbeat to a service a Uptimerobot or Cronitor

## Installation
```
composer require lotsofpixels/cronjobmonitor-magento2
```
## Version
Release 1.0.0

## Configuration
Stores -> Configuration -> Lotsofpixels -> Cronjob monitor
